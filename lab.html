✨ ANIMAÇÕES

Fade In

@keyframes fadeIn { from { opacity: 0; transform: translateY(-20px); } to { opacity: 1; transform: translateY(0); } } elemento { animation: fadeIn 0.8s ease-out; }

Scale Pulse

@keyframes scalePulse { 0%, 100% { transform: scale(1); } 50% { transform: scale(1.1); } } elemento { animation: scalePulse 2s ease-in-out infinite; }

Rotate

@keyframes rotate { from { transform: rotate(0deg); } to { transform: rotate(360deg); } } elemento { animation: rotate 3s linear infinite; }

/* Blur */ elemento { filter: blur(2px); } /* Saturate */ elemento { filter: saturate(2); } /* Brightness */ elemento { filter: brightness(1.5); } /* Combine */ elemento { filter: blur(1px) saturate(1.5) brightness(1.2); }

🚀 PRÓXIMOS PASSOS

1. Editar Cores Globais

Em `style.css`, `schedule.css` e `enter.css`, altere as variáveis CSS:

:root { --blood: #8B0000; --blood-bright: #CC0000; --neon-red: #ff0055; --neon-purple: #8800ff; --neon-green: #00ff00; }

2. Adicionar Novos Eventos

Em `schedule.html`, dentro de `.events-list`:

<div class="event"> <span class="time">HH:MM</span> <span class="event-name">Nome do Evento</span> <span class="event-desc">Descrição</span> </div>

3. Customizar Mensagens de Horror

Em `enter.js`, procure por `messages[]` e edite as strings:

const messages = [ 'Sua mensagem aqui', 'Outra mensagem de terror', 'Mais uma mensagem assustadora' ];

4. Adicionar Sons

Em `enter.js`, descomente e customize a função `playScarySound()`

5. Fazer Deploy

Veja o arquivo `DEPLOY.md` para instruções completas
