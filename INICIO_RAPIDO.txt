🆘 ENCONTROU UM PROBLEMA?

Efeitos não aparecem:
  → Limpe cache (Ctrl+Shift+R no Windows/Linux, Cmd+Shift+R no Mac)
  → Tente outro navegador
  → Abra DevTools → Console → procure por erros

Botões não funcionam:
  → Verifique se os arquivos HTML estão na mesma pasta
  → Confirme os nomes dos arquivos (exato, case-sensitive)

Cursos não aparece:
  → Tente outro navegador
  → Abra DevTools → Console → procure por erros

Som não toca:
  → Clique na página primeira
  → Navegadores modernos bloqueiam autoplay de áudio
